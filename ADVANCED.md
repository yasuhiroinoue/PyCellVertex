# Advanced Customization Guide

While `main.py` exposes the most common parameters via the Command Line Interface (CLI), researchers and developers who need to deeply alter the simulation's physics or topology can modify the core source code. 

This document serves as a map for deep customization.

## 1. Core Physical Constants (`constants.py`)

Simulation-wide physical and mathematical constants are hardcoded in `src/celldynamics/constants.py` to maintain strict parity with the original C++ implementation.

If you need to change the fundamental integration timescale or topological thresholds, edit this file directly:

- `DELTA_TIME` (Default: `1e-4`): The $\Delta t$ used for Euler integration. Changing this alters the relationship between simulation steps and physical time.
- `L_THRESHOLD` (Default: `1e-1`): The distance threshold at which two vertices are considered close enough to trigger a topological rearrangement (T1 transition or intersection).
- `L_RECONNECTED` (Default: `5e-2`): The initial length assigned to a newly created edge immediately after a T1 transition.
- `TIME_RECONNECT` (Default: `10.0`): The physical time interval between topology surgeries.

## 2. Modifying Force Equations (`force.py`)

If you want to introduce new physical phenomena (such as apoptosis, cell-substrate friction, or custom anisotropic stress), you will need to modify the equations in `src/celldynamics/force.py`.

- `calcAreaForce()`: Computes the force generated by cell area elasticity. Modify this if you want to change how cells resist compression or expansion.
- `calcLineForce()`: Computes the active and passive line tensions along cell edges. This is where the PCP (Planar Cell Polarity) oscillation term is currently implemented. If you want to change the tension dynamics or add completely new contractile forces, modify the logic here.

## 3. Initial Tissue Geometry (`init_plain.py`)

The default tissue is a regular honeycomb-like hex/quad grid generated dynamically.
If your research requires a specific initial state (e.g., a circular tissue patch, a wounded epithelium, or Voronoi noise):

- Modify `init_plain()` in `src/celldynamics/init_plain.py`.
- Ensure that the initial topological structures (`Vertex`, `Line`, `Cellula`) follow the strict counter-clockwise indexing rules required by the simulation engine.

## 4. Topological Surgeries (`rearrange.py` & `intersection.py`)

The simulation maintains tissue integrity through complex topological surgeries.

- **T1 Transitions & Intercalations**: Handled in `src/celldynamics/rearrange.py`.
- **T2/T3-like Boundary Intersections**: Handled in `src/celldynamics/intersection.py`.

**Warning:** Modifying the topology files is highly advanced. The vertex model requires exact, mathematically closed loops without dangling edges. Any custom implementation of a new intersection pattern must strictly preserve graph parity by properly deleting old boundary edges and appending new ones.
